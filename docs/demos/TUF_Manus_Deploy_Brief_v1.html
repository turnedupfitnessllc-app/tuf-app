ARCHITECTURE DECISIONS FOR MANUS
DecisionRecommended OptionWhy
Static Host Netlify (free tier) Drag-and-drop deploy, instant HTTPS, branch previews, free custom domain SSL, simplest for Marc's workflow
Admin Auth Netlify _headers basic auth No backend needed — single config file handles it. Upgrade to proper auth in Phase 2.
File Update Flow GitHub repo + Netlify auto-deploy Marc or Wilfred pushes updated files → live in 60s. No FTP, no manual upload.
localStorage Limitation Document + accept for Phase 1 Admin and member app must be open in same browser for live sync. Acceptable for internal use — full backend sync in Phase 2.
Phase 2 Backend Supabase (PostgreSQL + Auth + Realtime) Free tier generous, Realtime replaces localStorage sync globally, built-in auth replaces basic auth, row-level security for multi-trainer access. Wilfred will build API layer when ready.
Domain Structure app.turnedupfitness.com Separates app from marketing site. Admin at /admin. Staging at /staging.
